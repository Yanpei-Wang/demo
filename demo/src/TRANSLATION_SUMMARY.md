## Translation Guidelines

### Common Terms
- 导师 → Mentor
- 学员 → Mentee
- 轮次 → Round
- 会议 → Meeting
- Completed → Completed
- 进行中 → Active/In Progress
- 待开始 → Pending/Upcoming
- 当前 → Current
- 下一轮 → Next Round
- 即将开始 → Upcoming
- Personal Dashboard → Personal Dashboard
- 管理员 → Admin
- 用户 → User
- 角色 → Role
- 项目 → Program
- 参与 → Participation
- 数据 → Data
- 统计 → Statistics
- 分析 → Analysis

### Date Format
- Chinese: YYYY年MM月DD日
- English: Month DD, YYYY

### Status Labels
- 进行中 → Active
- Completed → Completed  
- 待开始 → Pending
- 已离职 → Terminated

### Button Labels
- 确认 → Confirm
- Cancel → Cancel
- 保存 → Save
- 关闭 → Close
- 查看 → View
- 编辑 → Edit
- 删除 → Delete
- 搜索 → Search
- 选择 → Select

### Validation Messages
- 请填写... → Please fill in...
- 最多... → Maximum...
- 最少... → Minimum...
- 不能超过... → Cannot exceed...
- 请至少... → Please at least...
